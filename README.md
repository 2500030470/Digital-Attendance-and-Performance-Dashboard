Project Description :

This project is a Java-based Digital Attendance System that manages student attendance using CSV files. It automates attendance recording and stores data using file handling with proper exception handling.

Features :

Reads user and attendance data from CSV files

Stores attendance records with date information

Uses ArrayList to manage attendance data

Handles file-related errors using exception handling

Reduces manual data entry

Technologies Used :

Java

Java Collections Framework

File Handling

CSV (Comma Separated Values)

GitHub

How to Run :

Open the project in VS Code or Eclipse

Ensure Java is installed on your system

Compile the program using javac

Run the main class using java AttendanceSystem

Files Used :

AttendanceSystem.java – Main class

AttendanceService.java – Attendance logic

CSVReader.java – Reads CSV files

input_data.csv – Input data

attendance_records.csv – Stores attendance records
